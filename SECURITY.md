# Security Policy

Please report security issues, accidental credential exposure, privacy issues,
or personal-data concerns privately by emailing `tbre@bath.ac.uk`.

Do not open a public GitHub issue for security vulnerabilities, leaked secrets,
or private personal information. Include a short description, affected files or
URLs, and enough detail for the maintainers to reproduce or verify the issue.

This is a static public website repository. The expected response is best
effort triage by the maintainers, prioritizing exposed secrets, private data,
and issues that affect the deployed site.
